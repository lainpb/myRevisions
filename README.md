# myRevisions

Este software tem por objetivo ajudar pessoas em suas revisões durante a aprendizagem de um idioma.

Você deve cadastrar todas as frases que você tem dificuldade para aprender.

Então o sistema irá apresentar uma frase por vez, em ordem da mais antiga para a mais nova.

Assim você poderá assimilar melhor o seu novo idioma fixando melhor na memória, as frases cadastradas.

Boa sorte.
